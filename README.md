# Nova Starter

Skeleton application for new Nova Projects.

## Requirements

- PHP 7.2+;
- Laravel 5.8+;
- Laravel Nova 2.0+;
- A valid Laravel Nova license.

## Getting Started

1. Create a fork of this repository;
2. Configure this application as any laravel project.
